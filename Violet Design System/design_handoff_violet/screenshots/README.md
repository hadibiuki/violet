# Screenshots

`00-gallery.png` is the design gallery overview.

For the most accurate per-screen reference, **open the live files** — static screenshots can't
capture the hover states, RTL mirroring, dark-mode toggle, or interactions:

- Open `demo.html` at the project root for a clickable gallery of all 11 screens.
- Or open any `templates/<slug>/<Name>.dc.html` directly in a browser.

(Open from the project tree so the fonts and compiled component bundle resolve.)

To regenerate static screenshots, open each template and capture full-page — but the live
prototypes are the authoritative visual reference.
